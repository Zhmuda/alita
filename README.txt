Бот вместе с БД расположен на сервере с IP 87.239.107.121 и портом 22. С помощью программы PuTTY происходит взаимодействие с сервером. 

Здесь я опишу стандартный процесс отключения или запуска бота:
1. Первым делом нас попросит ввести логин: пишем centos.
2. Далее переходим в папку с основным файлом бота: cd pythonProject1.
3. Далее нужно его выключить. Самый простой способ, которым пользуюсь я: sudo killall python3.
4. После загрузки новых файлов бота также стоит его включить. Важно помнить, что мы должны находиться в папке с main.py, и только после этого: nohup python3 main.py.

В коде также предусмотрена возможность запустить тестового бота. В файлe .env нужно раскомментировать строку с токеном с комментарием "тест" и заккоментировать другой токен. Также для удобства в конце программы стоит закомментировать рассылку по всем пользователям и оставить только свой ID, так уведомление придет только вам.

С помощью программы WINSCP можно загрузить новые файлы или выгрузить старые. Обычно я выгружаю БД для поиска ошибок.

Также стоит обратить внимание на файл nohup.txt в который записываются все выводы консоли.
